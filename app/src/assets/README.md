# app/src/assets/ - 静的ファイル

## 📂 このディレクトリの目的

開発時に使用する静的ファイル（ビルドに含まれるもの）を配置します。

---

## ✅ 何を配置するか

### 1. **画像ファイル**
- ロゴ・アイコン
- UI用の画像（背景、装飾等）
- 動画サムネイル（プレースホルダー）

### 2. **SVG**
- アイコン
- ロゴ

---

## ❌ 何を配置しないか

### **1. 動画ファイル**
❌ **動画は `assets/` に配置しません**

**理由:**
- OneTube MVPでは、動画は **Walrus分散ストレージ** に保存されます
- Seal経由でBLOB IDから動画URLを取得します
- `assets/`は開発時の静的ファイル（ビルドに含まれるもの）のみです

### **2. コンポーネント・ロジック**
- Reactコンポーネント → `App.tsx`
- ビジネスロジック → `lib/`

### **3. Backend関連**
- API endpoints → `server/`
