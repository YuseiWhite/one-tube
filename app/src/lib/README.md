# app/src/lib/ - 再利用可能なロジック

## 📂 このディレクトリの目的

複数箇所で使用される、UIに依存しない純粋なビジネスロジックとSDK統合を配置します。

---

## ✅ 何を配置するか

### 1. **Sui SDK統合**
- トランザクション構築
- Move関数呼び出しのラッパー
- Sui Client操作

### 2. **API Client**
- Backend APIとの通信
- HTTPリクエスト処理
- エラーハンドリング

### 3. **型定義**
- 共通で使用するTypeScript型
- APIレスポンスの型
- NFT・Sessionの型

### 4. **定数・設定**
- Package ID
- Kiosk ID
- 環境変数から読み込む設定値

### 5. **ビジネスロジック**
- データ変換・計算
- バリデーション
- フォーマット処理

---

## ❌ 何を配置しないか

### **1. UIコンポーネント**
❌ Reactコンポーネントは `App.tsx` に書く

### **2. Reactフック（MVP段階では不要）**

### **3. Backend処理**
❌ Express endpoints は `server/` に書く
